QR-Reader

An online tool for reading QR-Codes

About the application:

The reason that this application exists is a need for a QR-code scanner without having to install an app.
Basicaly you can open this site, agree to the terms once and start scanning QR-Codes.

Having trouble scanning QR-Codes?
Try disabling your camera flashlight.

You can open this application by browsing to:

  • marvin-brouwer.github.io/QR-Reader
  • tiny.cc/qrr

Specifications:

  • This web application requires access to your camera or filesystem
  • This web application uses cookies to store wether you've agreed to the terms

Creating QR-Codes:

If you want to easily generate QR-Codes, navigate to: goqr.me.
